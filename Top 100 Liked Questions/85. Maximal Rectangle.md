## Problem Description (Hard)
```
Given a 2D binary matrix filled with 0's and 1's, find the largest rectangle containing only 1's and return its area.
```

## Ideas
```
Two ways: stack or dynamic programming
```

## Solutions
```C++
class Solution {
public:

    int largestRectangleArea(vector<int>& heights) {
        
        heights.push_back(0);
        stack<int> aStack;
        int aResult = 0;
        for (int i=0; i<heights.size(); ++i) {
            while (!aStack.empty() && heights[aStack.top()] >= heights[i]) {
                int aTop = aStack.top();
                aStack.pop();
                
                int aSpan = i;
                if (!aStack.empty()) { aSpan -= (aStack.top() + 1); }
                aResult = max(aResult, aSpan * heights[aTop]);
            }
            aStack.push(i);
        }
        
        return aResult;   
    }
};
```
